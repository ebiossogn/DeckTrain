# DeckTrain — Instructions pour Claude Code

## Projet
Application web de formation interactive (alternative à PowerPoint/Canva)
Formation Tech pour équipe interne — Mai 2026

## Stack technique
- Next.js 14 (App Router)
- Tailwind CSS v3
- Framer Motion
- NextAuth.js
- Prisma + SQLite
- Tiptap (éditeur rich text)
- Zustand (state management)
- Recharts (graphiques)
- Shiki (coloration syntaxique)
- Lucide React (icônes)

## Règles importantes
- TOUJOURS afficher © CHRIST J. dans le footer de chaque page
- Dark mode par défaut avec toggle Light/Dark
- Couleur accent : Bleu cyan #00D4FF
- Animations Framer Motion sur tous les composants
- Police : Syne + Inter + JetBrains Mono
- Demander validation avant de passer à l'étape suivante

## Langue
- Code en anglais
- Commentaires en français
- UI en français
